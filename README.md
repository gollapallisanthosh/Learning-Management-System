# Learning Management System (LMS)

## Project Overview

The Learning Management System (LMS) is a web-based application designed to manage online courses, users, and learning resources. It allows students to enroll in courses, access learning materials, and track their progress, while administrators can manage courses, users, and content efficiently.

This project was developed as a final-year academic project to demonstrate practical skills in full-stack web development.

---

## Features

* User authentication (login and registration)
* Role-based access (Admin and Student)
* Course creation and management
* Student enrollment in courses
* View course content and materials
* Dashboard for tracking progress
* Responsive user interface

## Installation and Setup

### Prerequisites

* Node.js installed
* npm or yarn installed
* Database (MongoDB/MySQL) running

---

### Step 1: Clone the repository

```
git clone https://github.com/gollapallisanthosh/Learning-Management-System.git
cd Learning-Management-System
```

---

### Step 2: Install dependencies

#### For backend

```
cd server
npm install
```

#### For frontend

```
cd ../client
npm install
```

---

### Step 3: Run the application

#### Start backend

```
cd server
npm start
```

#### Start frontend

```
cd client
npm start
```
